## 💡 О проекте

**Wonder.ru** — это CRM + онлайн-запись, которая помогает частным мастерам (маникюр, стрижки, косметология, массаж, брови) перестать вести запись в блокноте и потерять меньше клиентов на no-show.

Мастер за 5 минут создаёт свою страницу `woner.ru/m/свой-ник`, делится ссылкой в соцсетях и на визитке. Клиент записывается сам в 2 клика, выбирая услугу и время. Бот напоминает о визите за 24 и 3 часа. Если клиент отменяет — слот автоматически освобождается.

## ✨ Что умеет

### Для мастера:
- 📅 Онлайн-запись 24/7 через MAX, Telegram или сайт мастера
- 🗂 CRM: клиентская база, история визитов, заметки
- 🎨 Портфолио с фото работ
- 💰 Прайс-лист с категориями услуг
- 📆 Гибкое расписание работы
- 📣 Рассылки клиентам (акции, новости)
- 🗺 Интеграция с Яндекс.Картами
- 📊 Аналитика: выручка, популярные услуги, загрузка

### Для клиента:
- 🔗 Запись в 2 клика по ссылке мастера
- 📆 Удобный календарь со свободными слотами
- 🔔 Напоминания в MAX/Telegram за 24 и 3 часа
- ❌ Отмена и перенос визита без звонков
- 🗺 Построение маршрута до мастера через Яндекс.Карты

## 🛠 Технологии

- **Backend:** Node.js 18+ · Express · PostgreSQL 15
- **Frontend:** React 18 · Vite · TailwindCSS
- **Мессенджеры:** MAX Bot API · Telegram Bot API
- **Cron / уведомления:** node-cron (каждые 10 минут)
- **Инфраструктура:** Docker · Docker Compose · Nginx · внешний SSL
- **Хранилище файлов:** S3-совместимое (загрузка портфолио и аватаров)

## 🚀 Быстрый старт (для разработчиков)
